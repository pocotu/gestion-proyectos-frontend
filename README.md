# gestion-proyectos-frontend

Scaffold del frontend con Vite + React para el Sistema de Gestión de Proyectos (MVP).

Instalación:

1. cd gestion-proyectos-frontend
2. npm install
3. npm run dev

Estructura mínima:
```
src/
  main.jsx
  App.jsx
  pages/
  components/
public/
package.json
README.md
```
